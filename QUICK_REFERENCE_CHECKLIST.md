## üîß Emergency Troubleshooting

### Stream Disconnects
1. Wait 30 seconds (auto-reconnect)
2. If not working: Enable mobile hotspot
3. Connect via USB tethering
4. Announce in Discord: "Brief interruption, returning shortly"

### No Microphone Audio
1. Check OBS mixer (not muted?)
2. Check mic plugged in and powered on
3. Check system settings (correct input device?)

### Screen Share Black
1. Verify correct display in source settings
2. Delete and re-add source
3. Try Window Capture instead

### Chat Not Visible
1. Refresh YouTube Studio
2. Use phone as backup to monitor chat
